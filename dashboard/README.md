# SentinelAI Dashboard

This is the Next.js + Tailwind dashboard for real-time alerts, incident timeline, and evidence preview.

## Features
- Live camera feed (MVP: placeholder)
- Real-time alert popups (WebSocket)
- Incident timeline
- Incident detail page
- Evidence preview
- Download PDF report
- Privacy blur toggle (face blur default ON)

## Getting Started
1. `npm install`
2. `npm run dev`
